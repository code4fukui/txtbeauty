# txtbeauty
日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

テキストファイルをUTF-8エンコーディングに変換します (バッチ変換をサポートしています)。

## デモ
https://code4fukui.github.io/txtbeauty/

## 機能
- テキストファイルをUTF-8エンコーディングに変換します
- 複数のファイルのバッチ変換をサポートしています
- オプション: JIS X 0213文字セットに縮小します

## 使い方
1. 変換したいテキストファイルをドラッグアンドドロップしてください。
2. 必要に応じて「JIS X 0213に縮小」オプションにチェックを入れて、JIS X 0213文字セットに制限することができます。
3. 変換されたテキストファイルは自動的にダウンロードされます。
